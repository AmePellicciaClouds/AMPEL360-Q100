# ATA 20-00 — Standard Practices — Airframe (General)

## Overview

This section covers the general aspects of standard practices for airframe maintenance on the AMPEL360 Q100 aircraft. It establishes the framework for procedures, techniques, and practices applicable to airframe structures.
