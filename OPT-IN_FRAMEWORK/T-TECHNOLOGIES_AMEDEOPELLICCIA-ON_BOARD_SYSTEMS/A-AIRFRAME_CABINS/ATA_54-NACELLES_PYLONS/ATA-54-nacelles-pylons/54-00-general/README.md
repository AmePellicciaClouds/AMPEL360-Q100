# ATA 54-00 — Nacelles/Pylons (General)

**ATA Chapter 54 | Sub-Subject: 54-00-general**

---

## Overview

This section covers the general aspects of nacelle and pylon structures for the AMPEL360 Q100 aircraft. It establishes the framework for nacelle/pylon design, construction, maintenance, and repair.

---

## Scope

| Area | Coverage |
|------|----------|
| Nacelles | Engine nacelle structures |
| Pylons | Wing-to-engine pylon structures |
| Cowlings | Fan cowls, thrust reverser cowls |
| Mounts | Engine mount structures |
| Maintenance | Inspection and repair procedures |
