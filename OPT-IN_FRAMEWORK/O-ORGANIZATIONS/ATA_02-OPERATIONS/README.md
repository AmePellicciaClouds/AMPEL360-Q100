# ATA 02 - OPERATIONS

**Operations Organization and Procedures**

## Description

ATA Chapter 02 defines the operational organization, procedures, and responsibilities for the AMPEL360 Q100 aircraft. This chapter establishes the framework for safe and efficient aircraft operations.

## Scope

This chapter includes:

- Operations organization structure and responsibilities
- Standard operating procedures (SOPs)
- Flight operations policies and guidelines
- Crew qualifications and training requirements
- Operational control procedures
- Flight planning and dispatch requirements
- Emergency response organization
- Safety management system (SMS) framework
